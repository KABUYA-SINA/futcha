# 🚀 Futcha — Discovery & Future Tech Platform

Plateforme dédiée à l’exploration des innovations en science et technologie.

---

## 🚀 Description

**Futcha** est une application web conçue pour explorer les avancées en science, technologie et innovations du futur.
Elle propose des contenus informatifs et inspirants visant à sensibiliser et informer les utilisateurs sur les technologies émergentes.

---

## 🎯 Objectifs du projet

* Créer une plateforme de contenu informatif
* Mettre en avant les innovations technologiques
* Proposer une interface moderne et engageante
* Structurer un site web responsive et accessible
* Améliorer l’expérience de lecture utilisateur

---

## 🛠️ Technologies utilisées

* REACT
* SASS
* SCSS
* JavaScript
* GitHub Pages (déploiement)

---

## ⚙️ Fonctionnalités

* 📰 Articles sur les technologies du futur
* 🌐 Navigation fluide entre les contenus
* 📱 Interface responsive
* 🎨 Design moderne et épuré
* ⚡ Chargement rapide des pages

---

## 🧠 Compétences développées

* Intégration front-end
* Création de contenu web structuré
* UI/UX design simple et efficace
* Déploiement via GitHub Pages
* Organisation de projet web statique
